SharePoint 2010 REST emulator

Using node.js and express 3.0.

About

The emulator is a node.js web server which serves static files (like this page) just like you would expect, but also responds to REST calls in the same way SharePoint does.

$.getJSON('/_vti_bin/listData.svc', function(data) {
   $('#result-root').text( JSON.stringify(data, null, 2) );
});

Here we've used jQuery to make an ajax request for a directory of all available lists '/_vti_bin/listData.svc'. Then we've taken the response, turned it into a string and "beautified" it a bit, then piped the results out.
